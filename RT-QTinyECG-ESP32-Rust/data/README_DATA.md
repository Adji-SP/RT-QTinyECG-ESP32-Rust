# Data Directory

This directory contains ECG signal data files used by the Python simulator
and GNUPlot visualization scripts.

## Files

### `sample_ecg.csv`

Generated by: `python python/generate_dummy_ecg.py`

**Format:**
```
time_ms,adc_value,label
```

| Column | Type | Description |
|---|---|---|
| `time_ms` | int | Timestamp in milliseconds since start |
| `adc_value` | int (0–4095) | Simulated 12-bit ADC ECG sample |
| `label` | int (0 or 1) | Ground truth: 0=Normal, 1=Abnormal |

**Notes:**
- Sampling rate: 250 Hz (4 ms per sample)
- Total duration: ~10 seconds (2500 samples)
- Normal ECG: PQRST-like waveform with Gaussian peaks
- Abnormal segment: Elevated baseline + irregular amplitude (simulated arrhythmia)

---

### `simulated_realtime_log.csv`

Generated by: `python python/realtime_ecg_simulator.py`

**Format:**
```
time_ms,adc_value,filtered_value,inference_us,prediction,alert,alert_latency_ms
```

| Column | Type | Description |
|---|---|---|
| `time_ms` | int | Timestamp in milliseconds |
| `adc_value` | int (0–4095) | Raw ADC sample |
| `filtered_value` | float | After moving average filter |
| `inference_us` | float | Inference duration in microseconds |
| `prediction` | int (0 or 1) | Classifier output: 0=Normal, 1=Abnormal |
| `alert` | int (0 or 1) | Alert state: 0=Off, 1=On |
| `alert_latency_ms` | float | Time from abnormal detection to alert activation |

This file is used as input for GNUPlot visualization scripts.

---

## Notes

- All CSV files use comma `,` as delimiter.
- Files are regenerated on each simulator run (overwritten).
- Do not commit large CSV files to version control. Add `data/*.csv` to `.gitignore` if needed.
- For real ESP32 hardware, capture serial output directly to a CSV file:
  ```bash
  # Linux/macOS
  cat /dev/ttyUSB0 | grep "^[0-9]" > data/hardware_log.csv
  ```

## Data Source

This project uses **simulated synthetic ECG data** for educational demonstration.
It does NOT use real patient ECG records.
For real ECG research data, see:
- PhysioNet (https://physionet.org/)
- MIT-BIH Arrhythmia Database
